# Release Plan

## MVP Release

Цель: доказать, что Electron-платформа подходит для gaze-игр.

Входит:

- каталог 14 игр с частью игр в статусе planned;
- общий gaze engine;
- общий session API;
- `Бабочки`;
- `Бульк`;
- `Цветы`;
- `Утки`;
- базовые настройки;
- экран результата.

## Therapy-ready Release

Цель: продукт можно использовать в занятиях со специалистом.

Входит:

- `Счёт`;
- `Выбери картинку`;
- `Съедобное`;
- WordBank;
- настройки профиля;
- метрики;
- research-informed подсказки.

## Full Games Release

Цель: все 14 игр без картинга доступны в Electron.

Входит:

- `Пирамидка`;
- `Рыбки`;
- `Прятки`;
- `Печать слов`;
- `Математика. Операции`;
- `Жаба`;
- `Теннис`.

## Polished Release

Цель: улучшить визуальную современность и качество занятий.

Входит:

- advanced адаптация сложности;
- красивые, но спокойные визуальные темы;
- расширенные отчёты;
- профили ребёнка;
- экспорт результатов;
- инструкции для специалиста;
- улучшенная калибровка и pre-flight check.
