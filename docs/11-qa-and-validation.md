# QA And Validation

## Общая проверка

- `npm run typecheck`.
- `npm run build`.
- Проверка запуска Electron.
- Проверка mouse fallback.
- Проверка Tobii status.
- Проверка resize window.
- Проверка выхода из каждой игры.

## Gaze QA

- При valid gaze курсор отображается стабильно.
- При invalid gaze игра не засчитывает выбор.
- Dwell progress сбрасывается при уходе взгляда.
- После выбора есть cooldown.
- Маленькие цели не используются на easy preset.
- Потеря gaze логируется как событие.

## Game QA

Для каждой игры проверить:

- старт;
- пауза;
- возврат в меню;
- завершение;
- повтор;
- Ошибка;
- подсказка;
- session metrics;
- отсутствие console errors;
- cleanup timers/requestAnimationFrame.

## Педагогическая проверка

- Понятна ли цель игры взрослому.
- Понятна ли инструкция ребёнку.
- Можно ли упростить игру перед запуском.
- Не перегружен ли экран.
- Есть ли короткий успешный цикл.
- Не звучит ли ошибка как наказание.

## Regression checklist

- Новая игра не ломает меню.
- Новая игра не ломает Tobii calibration page.
- Изменения gaze engine не ломают старые игры.
- WordBank errors не ломают весь каталог.
- Audio errors не ломают сессию.
