# Политика обезличенной аналитики

В packaged LINKa plays обезличенная аналитика включена всегда и не имеет opt-out. Перед первым выбором режима для каждой версии политики приложение показывает информационный диалог с кнопкой «Продолжить». Постоянная ссылка: <https://plays-metric.nkolinka.ru/privacy>.

## Что собирается

- случайный installation ID, не связанный с учётной записью или аппаратным ID;
- версия и build приложения, платформа, версия ОС и locale;
- app/game session UUID, route name и game ID из закрытого allowlist без query/search, режим меню и категория игры;
- числовая настройка dwell;
- lifecycle приложения, игры, Tobii и updater;
- агрегаты игровой сессии: активная длительность, pause, шаги, успехи, ошибки, подсказки, отмены, потери взгляда, изменения сложности и раздельные sample counts;
- fingerprint ошибки без исходного текста.

## Что запрещено

- имя, контакты и другие данные пользователя;
- текст, фразы, слова, expected/actual ответы;
- непрерывные или отдельные координаты взгляда и указателя, timestamp pointer sample;
- `targetId`, board/FEN и другие снимки игрового состояния;
- пути файлов, error message и stack;
- raw Tobii diagnostics.

Raw diagnostics остаётся только локальной функцией отчёта и копирования в буфер. Клиент не содержит IPC/API для её удалённой загрузки.

## Хранение

Сервер хранит принятые данные бессрочно, без автоматического TTL. Удаление по запросу не предусмотрено. До доставки записи находятся в bounded spool приложения; сеть никогда не блокирует игру. Доставка at-least-once, поэтому backend должен считать `event_id` идентификатором повторной записи.
