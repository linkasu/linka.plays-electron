# Research Track

Этот раздел фиксирует научную основу для игр взглядом. Он не заменяет работу дефектолога, но задаёт ограничения и практические правила для разработки.

## Как использовать

- При проектировании игры сначала определить навык: фиксация, слежение, выбор, классификация, AAC, счёт, буквы.
- Затем проверить `design-principles-from-research.md`.
- Для спорных решений смотреть `eye-tracking-bibliography.md`.
- Любую метрику формулировать как наблюдение, а не диагноз.

## Главный вывод

Eye-tracking в играх полезен, когда он:

- снижает моторный барьер;
- даёт ребёнку способ выбора;
- тренирует внимание короткими повторяемыми заданиями;
- поддерживает коммуникацию;
- даёт взрослому наблюдаемые метрики;
- не перегружает ребёнка и не превращает занятие в тест.
