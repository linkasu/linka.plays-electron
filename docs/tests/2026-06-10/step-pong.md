## Правила и проигрыш

| Риск | Признак | Доказательство | Рекомендация |
|---|---|---|---|
| high | Ручная находка по правилам | Мяч не теряется: неправильная позиция только включает подсказку. | wrong lane = missed ball/минус жизнь, lives=0 = loss, серия отбиваний = win. |
| medium | Ошибки не завершают сессию | finishOnMistakes: false | Допустимо для терапии, риск для стратегий/головоломок. |
| medium | Ошибка превращается в подсказку | mistake + hint | Для учебных игр нормально, для партийных игр может быть багом правил. |
| medium | Есть safe/reset/fallback механизм | reset/createInitial/clearTopRows/gentleFallback/softReturn/safe | Проверить, не маскирует ли он поражение. |
| high | Нет явного outcome партии | не найден winner/loss/draw/gameOver | Стратегической игре нужен win/loss/draw или documented trainer mode. |
