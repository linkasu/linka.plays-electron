# Сольфеджио — runtime audit seed

## Проверка

- Route: `/games/solfege`
- Основные цели: белые клавиши одной октавы на пианино.
- Сцена: интерактивное пианино без текстовых подписей на клавишах, прогресс нот, optional звук.

## QA

- Проверить в Electron CDP, не только в browser tab.
- На 800×600 пианино должно быть видимо без нижнего ряда кнопок и без скролла.
- Звук включается вручную и остаётся тихим.
- Базовый цикл: прозвучала случайная нота, ребёнок повторил её на клавише.
- Ошибка не даёт резкого звука.
