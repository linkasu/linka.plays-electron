## Описание из реестра

Ставь шарики на мини-поле 5×5 и спокойно собирай линии из 3–5 одинаковых цветов; неверный выбор только включает подсказку.

Self-menu: Собери линию шариков.

## Правила и проигрыш

| Риск | Признак | Доказательство | Рекомендация |
|---|---|---|---|
| high | Ручная находка по правилам | Поле при заполнении очищается без проигрыша, подсказочные ходы видны постоянно. | Полное поле без линии должно давать loss; suggestedMoves показывать только после запроса/ошибки. |
| medium | Ошибки не завершают сессию | finishOnMistakes: false | Допустимо для терапии, риск для стратегий/головоломок. |
| medium | Описание закрепляет беспроигрышность | текст «без ошибок/поражения/любой выбор/только подсказ» | Проверить соответствие категории игры. |
| medium | Ошибка превращается в подсказку | mistake + hint | Для учебных игр нормально, для партийных игр может быть багом правил. |
| medium | Есть safe/reset/fallback механизм | reset/createInitial/clearTopRows/gentleFallback/softReturn/safe | Проверить, не маскирует ли он поражение. |
| high | Нет явного outcome партии | не найден winner/loss/draw/gameOver | Стратегической игре нужен win/loss/draw или documented trainer mode. |

## Рекомендации

- Проверить игру в реальном Electron окне через CDP, если UI-риск средний или высокий.
- Для category=strategy требовать явный outcome win/loss/draw либо документировать, что это тренажер, а не партия.
- Не считать статический аудит визуальным подтверждением: он фиксирует места для ручной/рантайм проверки.
