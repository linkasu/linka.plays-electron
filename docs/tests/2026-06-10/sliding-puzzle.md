| Риск | Признак | Доказательство | Рекомендация |

| medium | Ошибка превращается в подсказку | mistake + hint | Для учебных игр нормально, для партийных игр может быть багом правил. |
| medium | Есть safe/reset/fallback механизм | reset/createInitial/clearTopRows/gentleFallback/softReturn/safe | Проверить, не маскирует ли он поражение. |

## Рекомендации

- Не считать статический аудит визуальным подтверждением: он фиксирует места для ручной/рантайм проверки.
