# Цветы

## Unity source

- Scene: `Assets/Games/Flowers/Flowers.unity`.
- Asset: `Assets/Resources/Games/Games/Flowers.asset`.

## Core mechanic

Ребёнок удерживает взгляд на бутоне. Бутон растёт и раскрывается в цветок. Успех создаёт сад.

## Игровой цикл

```text
бутон появился -> удержание взгляда -> цветок растёт -> цветок раскрылся -> сад пополнился
```

## Управление взглядом

- Dwell target с визуальным ростом.
- Чем дольше gaze stay, тем больше бутон.
- При уходе progress может немного откатываться, но не мгновенно.

## Дефектологическая ценность

- Удержание фиксации.
- Терпение и стабильность.
- Причинно-следственная связь.
- Низкая когнитивная нагрузка.

## Современное переосмысление

Сделать `сад успехов`: каждый успешный dwell добавляет цветок. В конце ребёнок видит свой сад, а не сухой счёт.

## Уровни сложности

- Gentle: один большой бутон, медленный рост, dwell 1500ms.
- Standard: несколько мест для цветов, dwell 1000ms.
- Challenge: выбрать цветок нужного цвета или формы.

## Метрики

- Успешные цветы.
- Средняя длительность удержания.
- Количество срывов фиксации.
- Нужные подсказки.

## MVP

- Один бутон за шаг.
- Dwell grow animation.
- Итоговый сад.

## Therapy-ready

- Настройка отката progress.
- Настройка dwell.
- Подсказки при частом уходе взгляда.

## Polished

- Сезоны/темы.

## QA checklist

- Цветок не выбирается мгновенно без dwell.
