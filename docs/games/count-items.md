## Core mechanic

Показано несколько предметов. Ребёнок выбирает нужное число; повторная попытка оставляет тот же набор и не раскрывает ответ.

## Игровой цикл

```text
предметы появились -> ребёнок считает -> выбирает число -> feedback -> новый набор
```

## Управление взглядом

- Dwell target на числовых карточках.
- Предметы не должны быть gaze targets в базовом режиме.
- Можно подсвечивать предметы по одному в режиме обучения.

## Дефектологическая ценность

- Количественное восприятие.
- Сопоставление количества и цифры.
- Визуальная группировка.
- Выбор ответа без речи.

## Современное переосмысление

Сделать предметы тематическими: яблоки, рыбки, цветы. В режиме обучения ребёнок может взглядом `пересчитать` предметы: каждый gaze enter мягко подсвечивает предмет.

## Уровни сложности

- Gentle: 1-3 предмета, 2 варианта ответа.
- Standard: 1-9 предметов, 3-4 варианта.
- Challenge: группировки, distractor layout, без подсказок.

## Метрики

- Accuracy по числам.
- Время ответа.
- Ошибки рядом с правильным числом.
- Использование подсчёта/подсказки.
