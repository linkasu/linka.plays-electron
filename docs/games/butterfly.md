# Бабочки

## Unity source

- Scene: `Assets/Games/Butterfly/Butterfly.unity`.
- Asset: `Assets/Resources/Games/Games/Butterfly.asset`.
- Current Electron MVP: `src/frontend/games/butterfly/ButterflyGame.vue`.

## Core mechanic

Ребёнок двигает взглядом по экрану, а игра оставляет след из бабочек. Это игра не на правильный ответ, а на ощущение контроля и движение взглядом.

## Игровой цикл

```text
взгляд движется -> появляется след -> бабочки оживают -> ребёнок видит результат движения
```

## Управление взглядом

- Используется continuous gaze pointer.
- Не нужен dwell click для базового режима.
- При invalid gaze след временно не создаётся.
- Можно показывать точку текущего взгляда.

## Дефектологическая ценность

- Слежение взглядом.
- Осознание связи движения взгляда и результата.
- Низкий стресс, свободный исследовательский выбор.
- Подходит как первая игра после калибровки.
