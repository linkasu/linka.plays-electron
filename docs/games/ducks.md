# Утки

## Unity source

- Scene: `Assets/Games/Ducks/Ducks.unity`.
- Asset: `Assets/Resources/Games/Games/Game.asset`.
- Unity name: `Утки`.

## Core mechanic

Найти утку и выбрать её взглядом. После выбора утка исчезает или уплывает, засчитывается шаг.

## Игровой цикл

```text
утка появилась -> ребёнок смотрит -> dwell завершён -> утка уплыла -> следующая утка
```

## Управление взглядом

- Dwell target на утке.
- Магнит к центру утки в gentle preset.
- Большая invisible hit area вокруг sprite.

## Дефектологическая ценность

- Визуальный поиск.
- Выбор объекта.
- Тренировка целевого взгляда.
- Переход от одной цели к другой.

## Современное переосмысление

Сделать пруд с утками. На базовом уровне выбирается любая утка. Позже задача становится конкретной: `найди жёлтую`, `найди большую`, `найди утку с шапкой`.

## Уровни сложности

- Gentle: одна крупная утка.
- Standard: 3 утки, выбрать подсвеченную/названную.
- Challenge: утки медленно двигаются, появляются distractors.

## Метрики

- Время поиска.
- Успешные выборы.
- Ошибочные выборы.
- Отмены dwell.

## MVP

- Статичные утки.
- Dwell выбор.
- 5-10 шагов сессии.

- Ошибочный выбор не завершает уровень резко.
