# Математика. Операции

## Unity source

- Scene: `Assets/Games/Math/Math.unity`.
- Asset: `Assets/Resources/Games/Games/MathActions.asset`.
- Script: `MathController.cs`.

## Core mechanic

Показан арифметический пример. Ребёнок вводит ответ через gaze-friendly numpad.

## Игровой цикл

```text
пример показан -> ребёнок выбирает цифры -> проверка -> feedback -> новый пример
```

## Управление взглядом

- Dwell target на цифрах.
- Кнопки `стереть`, `проверить`, `повторить` крупные.
- Для multi-digit ответа нужен confirm или явная кнопка `Готово`.

## Дефектологическая ценность

- Арифметика.
- Рабочая память.
- Пошаговый ввод.
- Самостоятельный ответ без речи/руки.

## Современное переосмысление

Не начинать с всех операций. Сделать режимы: сложение до 5, сложение до 10, вычитание, умножение, деление. Добавить визуальные опоры: предметы, number line, разложение.

## Уровни сложности

- Gentle: сложение 1-5, один знак, визуальная опора.
- Standard: сложение/вычитание до 20.
- Challenge: умножение/деление, multi-digit ответы.
