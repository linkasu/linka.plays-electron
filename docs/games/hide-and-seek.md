# Прятки

## Unity source

- Scene: `Assets/Games/Hide-and-seek/Hide-and-seek.unity`.
- Asset: `Assets/Resources/Games/Games/Hide-and-seek.asset`.

## Core mechanic

Ребёнок ищет спрятанный объект или персонажа на картинке и выбирает его взглядом.

## Игровой цикл

```text
сцена показана -> инструкция кого найти -> взгляд ищет -> найденный объект выбран -> персонаж появляется
```

## Управление взглядом

- Dwell target на скрытом объекте.
- Подсказка может расширять область или подсвечивать сектор.
- Визуальный шум регулируется preset.

## Дефектологическая ценность

- Визуальный поиск.
- Сканирование экрана.
- Понимание инструкции.
- Переключение внимания.

## Современное переосмысление

Сделать сцены как уютные иллюстрации: комната, лес, сад. Объекты не должны быть слишком мелкими. Прятки должны быть радостным поиском, а не тестом на остроту зрения.

## Уровни сложности

- Gentle: объект крупный, часть видна, фон простой.
- Standard: объект частично скрыт, 2-3 distractors.
- Challenge: больше distractors, словесная инструкция по признаку.

## Метрики

- Время поиска.
- Количество подсказок.
- Ошибочные выборы.
- Сектора экрана, где чаще теряется взгляд.

## MVP

- Несколько статичных сцен.
- Один target на сцену.
- Подсказка после timeout.

## Therapy-ready

- Уровни визуального шума.
- Инструкции по признакам.
- Режим `теплее/холоднее`.

## Polished

- Маленькая история: найти друзей.
- Персонажи реагируют на найденность.
- Коллекция найденных объектов.

## QA checklist

- Объект не слишком мал.
- Подсказка не выдаёт ответ мгновенно на challenge.
- Фон не мешает gaze tracking.
