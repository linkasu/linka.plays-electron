## Назначение

Ребёнок собирает два бутерброда по рецепту: выбирает ингредиенты в заданной последовательности, видит слои на тарелке и завершает каждый рецепт верхним хлебом.

Самостоятельная формулировка для меню: Собери бутерброд по рецепту.

## Игровой цикл

```text
показ рецепта и текущего шага -> выбор ингредиента -> слой появляется на тарелке -> следующий шаг, следующий бутерброд или результат
```

## Управление взглядом

- Основной ввод должен быть gaze-first; mouse fallback сохраняется для отладки и занятий без трекера.
- Минимальный целевой размер из registry: 150 px.
- Базовый dwell из registry: 1300 ms.
- Последний Electron CDP audit: failures = 0, max targets = 5, min visible targets = 5.
- Минимальная short-side эвристика targets: 0.188.

## Дефектологическая ценность

- тренирует понимание порядка и причинности
- поддерживает пошаговую инструкцию
- снижает нагрузку через крупный выбор

## Метрики и сессия

- Сессия должна использовать общий game session flow и завершаться через результат для взрослого.
- Важные метрики: успешные выборы, ошибки/отмены, dwell time, подсказки, потеря валидного взгляда.
- Рекомендуемая длительность занятия: 130 сек.

## Audio/TTS

- Игра включает тихий звук по умолчанию.
- TTS даёт нейтральный стартовый prompt и короткие реакции на успех, ошибку, готовый рецепт и завершение.
- Пока звучит TTS/feedback, выбор ингредиентов заблокирован.
- Ошибка не подсвечивает правильный ингредиент и не раскрывает его отдельной подсказкой.

## QA checklist

- Route открывается в Electron, не только в обычном браузере.
- На 800×600 и 1024×600 основные цели видны без ручного скролла.
- HUD, prompt и overlays не перекрывают активные цели.
- Для canvas/fullscreen игры PNG подтверждает непустую сцену и читаемые active zones.
- Звук, если есть, остаётся опциональным и не ломает gameplay при ошибке загрузки.
- Для игр с правилами model tests покрывают правильный ответ, ошибки и завершение раунда.
- Правильный ингредиент не должен подсвечиваться до выбора.
- Ошибка должна быть нейтральной и не подсказывать конкретный правильный ингредиент.
