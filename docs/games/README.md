# Games

Игровые документы описывают перенос всех Unity-игр, кроме `Картинг/Лабиринт`.

## Общий шаблон

Каждая игра описывается через:

- Unity source.
- Core mechanic.
- Игровой цикл.
- Управление взглядом.
- Дефектологическая ценность.
- Современное переосмысление.
- Уровни сложности.
- Метрики.
- MVP.
- Therapy-ready.
- Polished.
- QA checklist.
