# Пирамидка

## Unity source

- Scene: `Assets/Games/Pyramid/Pyramid.unity`.
- Asset: `Assets/Resources/Games/Games/Pyramid.asset`.
- Script: `TorusController.cs`.

## Core mechanic

Собрать пирамидку в правильном порядке. Ребёнок выбирает кольца взглядом, игра укладывает их на стержень.

## Игровой цикл

```text
кольца лежат -> выбрать следующее -> кольцо переместилось -> следующий шаг -> пирамидка собрана
```

## Управление взглядом

- Dwell target на кольцах.
- На easy подсвечивать правильное кольцо.
- На standard показывать правило: от большого к маленькому.

## Дефектологическая ценность

- Последовательность.
- Сравнение размера.
- Планирование.
- Понимание порядка.

## Современное переосмысление

Не завязываться на имена объектов `0..3`, как в Unity. Генерировать модель колец явно: размер, цвет, правильный индекс.

## Уровни сложности

- Gentle: 3 кольца, правильное подсвечено.
- Standard: 4-5 колец, порядок по размеру.
- Challenge: порядок по цветовой подсказке или смешанные признаки.

## Метрики

- Правильные шаги.
- Ошибки порядка.
- Время выбора каждого кольца.
- Подсказки.

## MVP

- 4 кольца.
- Dwell выбор.
- Автоматическая анимация укладки.

## Therapy-ready

- Настройка количества колец.

## Polished

- Ошибка не ломает последовательность.
