# Рыбки / дно

## Unity source

- Scene: `Assets/Games/Fishes/Fishes.unity`.
- Asset: `Assets/Resources/Games/Games/Fishes.asset`.
- Unity name: `дно`, но в Electron лучше назвать `Рыбки`.

## Core mechanic

Ребёнок смотрит на рыбку или следует за ней взглядом. После удержания крючок подцепляет рыбку и вытягивает её вверх за край экрана.

## Игровой цикл

```text
рыбка появилась -> взгляд поймал рыбку -> крючок вытягивает рыбку вверх за экран -> засчитано -> новая рыбка
```

## Управление взглядом

- В базовом режиме dwell на статичной рыбке.
- В развитом режиме smooth pursuit рядом с движущейся рыбкой.
- Effective target должен быть крупнее видимой рыбки.

## Дефектологическая ценность

- Слежение взглядом.
- Переключение внимания.
- Удержание цели в поле зрения.
- Снижение страха перед движущимися объектами через скорость.

## Современное переосмысление

Сделать подводное пространство. Рыбки не должны мелькать. Задачи: `посмотри на рыбку`, `проводи рыбку до коралла`, `найди рыбку нужного цвета`.

## Уровни сложности

- Gentle: одна крупная рыбка.
- Standard: 2-3 рыбки, простая инструкция.
- Challenge: следование по маршруту, переключение между рыбками.

## Метрики

- Время удержания на рыбке.
- Время рядом с движущейся целью.
- Потери gaze.
- Успешные переключения.

## MVP

- Одна рыбка как gaze target.
- Анимация при выборе.
- Session progress.

## Therapy-ready

- Настройка скорости.

## Polished
