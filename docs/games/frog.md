# Жаба

## Unity source

- Scene: `Assets/Games/Frog/Frog.unity`.
- Asset: `Assets/Resources/Games/Games/Frog.asset`.
- Scripts: `DragonSpawn.cs`, `DragonMoveController.cs`, `StrechToController.cs`.

## Core mechanic

Жаба ловит движущиеся цели взглядом. В Unity это драконы/объекты, в Electron лучше сделать более дружелюбный образ: мошки, жучки или светлячки.

## Игровой цикл

```text
цель летит -> ребёнок смотрит -> язык тянется к цели -> цель поймана -> следующая цель
```

## Управление взглядом

- Moving gaze target.
- В gentle режиме цель движется медленно или останавливается при gaze enter.
- Dwell короткий, но не мгновенный.
- Target magnet полезен.

## Дефектологическая ценность

- Реакция.
- Переключение внимания.
- Координация взгляда с движущейся целью.
- Timing, но без стрессового таймера на базовом уровне.

## Современное переосмысление

Снизить агрессию и скорость. Сделать жабу помощником, которая ловит светлячков. Ошибка означает `улетел`, а не `провал`.

## Уровни сложности

- Gentle: один медленный светлячок.
- Standard: цели появляются с разных сторон.
- Challenge: несколько целей, разные цвета, ускорение только после успехов.

## Метрики

- Время реакции.
- Успешные поимки.
- Улетевшие цели.
- Потери gaze на движущейся цели.

## MVP

- Один тип цели.
- Dwell capture.
- Простое движение.

## Therapy-ready

- Настройка скорости.
- Настройка размера цели.
- Режим без штрафа за пропуск.

## Polished

- Мягкая анимация языка.
- Ночная поляна со светлячками.
- Серии с разными паттернами движения.

## QA checklist

- Цель не становится слишком быстрой.
- Capture zone соответствует видимому объекту.
- При пропуске игра не создаёт стрессовую серию ошибок.
