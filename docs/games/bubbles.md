# Бульк

## Unity source

- Scene: `Assets/Games/Bubbles/Bubbles.unity`.
- Asset: `Assets/Resources/Games/Games/Bubbles.asset`.
- Unity name: `Бульк`.
- Category: `Знакомство с трекером`.

## Core mechanic

На экране появляются пузыри. Ребёнок смотрит на пузырь, удерживает взгляд, пузырь заполняет dwell progress, лопается, звучит мягкий звук воды, засчитывается шаг.

## Игровой цикл

```text
пузырь появился -> взгляд вошёл -> progress заполняется -> пузырь лопнул -> новый пузырь
```

## Управление взглядом

- Основной ввод: dwell target.
- Mouse fallback: hover/click или удержание курсора.
- Easy режим: большие пузыри, один активный пузырь.
- Challenge режим: несколько пузырей, но активная цель всё равно визуально понятна.

## Дефектологическая ценность

- Первичная тренировка фиксации.
- Причинно-следственная связь: `смотрю -> происходит действие`.
- Безопасное знакомство с dwell без когнитивной нагрузки.

## Современное переосмысление

Не копировать статичные Unity bubbles. Сделать спокойный водный экран, где пузыри появляются в разных местах, но не слишком близко к краям. При успехе вода мягко расходится кругами.

## Уровни сложности

- Gentle: 1 большой пузырь, dwell 1400ms, без отвлекающих объектов.
- Standard: 2-3 пузыря, dwell 1000ms, цель подсвечивается при gaze enter.
- Challenge: 4-5 пузырей, часть пузырей медленно движется, dwell 750-900ms.

## Метрики

- Успешные пузыри.
- Отмены dwell.
- Среднее время до лопания.
- Потери gaze.
- Предпочитаемые зоны экрана.

## MVP

- Canvas или SVG пузыри.
- Dwell progress.
- Session steps.
- Result screen.

## Therapy-ready

- Настройки размера пузырей.
- Настройки dwell.
- Режим `один пузырь`.
- Подсказка взрослому при частых отменах dwell.

## Polished

- Живой водный фон без перегруза.
- Разные типы пузырей.
- Награда-сцена: аквариум становится ярче.
- Мягкие аудио-варианты.

## QA checklist

- Пузырь не лопается при invalid gaze.
- Dwell сбрасывается при уходе.
- После лопания есть cooldown.
- Пузырь не появляется под HUD.
