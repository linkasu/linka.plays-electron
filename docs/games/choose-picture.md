# Выбери картинку

## Unity source

- Scene: `Assets/Games/ChoosePicture/ChoosePicture.unity`.
- Asset: `Assets/Resources/Games/Games/ChoosePicture.asset`.
- Script: `ChoosePictureController.cs`.

## Core mechanic

Игра задаёт вопрос: `Где яблоко?` Ребёнок выбирает картинку/emoji взглядом. Возможен обратный режим: показана картинка, надо выбрать слово.

## Игровой цикл

```text
звучит вопрос -> варианты показаны -> ребёнок выбирает -> feedback -> новый вопрос
```

## Управление взглядом

- Dwell target на карточках ответов.
- Карточки большие и далеко друг от друга.
- После gaze enter карточка мягко увеличивается.

## Дефектологическая ценность

- Receptive vocabulary.
- AAC-like выбор.
- Понимание инструкции без речи.
- Связь слова, звука и изображения.

## Современное переосмысление

Сделать карточки современными: крупный emoji/картинка, слово под ним, clear feedback. Взрослый может выбрать категории слов.

## Уровни сложности

- Gentle: 2 варианта, familiar words.
- Standard: 3-4 варианта.
- Challenge: близкие категории или обратный режим `картинка -> слово`.

## Метрики

- Правильные ответы.
- Ошибочные ответы.
- Время выбора.
- Категории с трудностями.
- Повторы аудио.

## MVP

- WordBank JSON.
- 2-4 карточки.
- Dwell выбор.

## Therapy-ready

- Выбор категорий.
- Настройка количества вариантов.
- Повтор вопроса.
- Режим emoji/слово.

## Polished

- Красивые карточки с иллюстрациями.
- Озвучка вопроса и ответа.
- Серии по темам: еда, животные, дом.

## QA checklist

- Правильный ответ всегда есть среди вариантов.
- Варианты не повторяются, если хватает слов.
- При нехватке слов игра показывает понятную ошибку взрослому.
